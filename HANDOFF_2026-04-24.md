# AE8 — World-Class Redesign Handoff
**Date:** 2026-04-24
**Branch:** `redesign/world-class`
**Predecessor tag:** `v0.1-original-design` (commit `e6a0efa`)
**Governance base:** `ef55f97` (master, 2026-04-23 governance commit)

## What changed
Complete visual overhaul of `index.html` plus `sw.js` cache bump.
Data, asset inventory, user-facing functionality preserved.

### Aesthetic direction
**Brutalist Athletic Editorial.** Black + hot pink `#FF0099`, condensed
athletic display type (Big Shoulders Display), technical mono accents
(JetBrains Mono), body set in DM Sans. Sharp geometric cards (no border
radius on most surfaces), ticker marquee, magazine-style issue metadata,
trading-card profile view.

### What's preserved (byte-accurate or near)
- **DRIP_DATA** — all 70 products across 8 categories, all external image
  URLs and `ae8approved` flags intact.
- **PLAYLIST** — all 21 audio tracks with `src` paths unchanged.
- **CAMPS** — all 109 camp entries (extracted programmatically from
  v0.1's hardcoded HTML into a structured JS array, `date/title/sub/loc/url`
  tuples). Filter logic ported — matches v0.1 text-based rules.
- **tournaments[]** — all 49 7v7 events.
- **popWarnerTeams[]** — all 45 Pop Warner champion/finalist teams.
- **COORDS{}** — city → `[lat,lng]` lookup, 100+ cities.
- **Music player engine** — shuffle, iTunes preview fallback, YouTube
  embed fallback, auto-advance — logic unchanged, only DOM selectors
  adapted for new mini-player markup.
- **Auth** — localStorage-based sign-in/sign-up/edit/sign-out; same data
  model (`ae8_users`, `ae8_session`, `ae8_highlights_<username>`).
- **Film Room** — IntersectionObserver autoplay, upload via
  `URL.createObjectURL`, like/share, highlight persistence.
- **Map** — CartoDB dark tiles, three layer groups (camps/tournaments/
  popwarner) with pink/white/gold pins; popups restyled in brand voice.

### Functional additions in redesign
- **Ticker marquee** — rolling masthead ticker with brand + drop + camp
  callouts (CSS animation, paused by `prefers-reduced-motion`).
- **Editorial masthead** — sticky header with issue meta (`N°04 · VOL.01
  · BUILT DIFFERENT`), pulsing brand dot, avatar that reflects auth state.
- **Hero film** — 4:5 aspect, full-bleed, `LIVE` + `REEL 04` badges.
- **Home "Navigate" quick grid** — numbered brutalist tiles (01/02/03/04).
- **Camps `NEXT UP` marker** — first upcoming camp gets a pink stripe
  and ribbon.
- **Map filter chips** — All / Camps / 7v7 / Pop Warner toggle layer
  visibility.
- **Film modal** — tapping a profile highlight opens full-screen player.
- **Player card locker** — trading-card layout with position/grade/clips
  stats grid.
- **Past camps purged** — only upcoming sessions render (by date).
- **Staggered entry animations** — `.stagger` class drives 60ms-delay
  reveals for grids.
